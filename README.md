# Scientific Graphing Calculator

An advanced scientific graphing calculator built with Streamlit for plotting experimental data and performing regression analysis.

## Features

- Plot scientific graphs
- Calculate line of best fit
- Compute slope and intercept
- Generate equation of graph
- Calculate R² value

## Tech Stack

- Python
- Streamlit
- NumPy
- Plotly
- Scikit-Learn

## Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/scientific-graphing-calculator.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
